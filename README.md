# Project setup Windows

1. Install npm 

You can get npm by installing Node.js

2. Install the following NPM packages in the command prompt with admin permission

npm install -g typescript

npm install -g http-server

3. Run the starter code make-menger.py

If you are on windows please install python version 3.11 so that distills works.
3.11.9 has a install on the website for simpliest download.

You should get a new directory: /dist

4. From the project root directory that contains /dist run the following in the terminal

http-server dist -c-1

5. You should get a message saying that the server started and you can launch the app by clicking on http://127.0.0.1:8080
